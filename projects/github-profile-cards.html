Github Profile Cards

Github Repository

Overview

This application is designed to showcase Github profile information. It was built with raw HTML, CSS, and JS, and uses It uses Github's API to fetch and display user details. Users can view public repositories, followers, and people they are following.

Development

This project served as a valuable learning experience in working with APIs and becoming familiar with raw HTML, CSS, and JS. The development process was smooth, with only minor changes to the appearance of the user cards being made.

Features

The application presents a user's GitHub profile information in a visually appealing format similar to an Instagram profile layout.

Challenges

There were no real challenges of this project; instead, it served as a learning opportunity to understand APIs.

Results

From this project, I learned to use APIs and became better at developing web applications without relying on frameworks like React.

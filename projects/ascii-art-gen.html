ASCII Art Generator

Github Repository

Overview

This program converts an image to ASCII art using Python and the Pillow(PIL) Library. Users can input an image they want to convert to ASCII art and the program will output an ascii_image.txt file with the ASCII art.

Development

This project was made with Python and served as a way to become familiar with libraries in Python and using a Python virtual environment(venv). A Python venv can be setup easily via Windows Power Shell by running:
python -m venv c:\path\to\myenv The development process went smoothly and helped me to learn more about virtual environments in Python.

Features

The program converts a user's uploaded image to ASCII art and outputs a .txt file containing the art.

Challenges

There were not really any challenges of this project; instead, it served as a learning opportunity to work with Python in a venv and use the Pillow library.

Results

This project taught me about using virtual environments for Python and refined my ability to use different Python libraries.
